# Netflix Data Analysis using Power BI

## Project Overview:
This project analyzes Netflix movies and TV shows data using Power BI to identify trends, patterns, and insights related to content distribution and growth.

## Objective:
- Analyze Movies vs TV Shows
- Identify content trends over years
- Explore genre and rating distribution
- Analyze country-wise content availability

## Tools & Technologies:
- Power BI
- DAX
- CSV Dataset
- Data Cleaning & Transformation

## Key Insights:
- Movies form the majority of Netflix content
- Content production increased significantly after 2015
- Drama and International content dominate the platform
- USA has the highest content count
